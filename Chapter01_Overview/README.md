# Chapter 01: Overview

本章是《PyTorch深度学习实践》的课程导论，主要用于建立对课程整体路线、深度学习基本流程和 PyTorch 学习方式的初步认识。

## 本章目标

学习完本章后，需要先形成几个整体印象：

- 深度学习不是单独的一段代码，而是一套从数据、模型、损失函数到优化算法的完整流程。
- PyTorch 是用来构建、训练和调试深度学习模型的工具，后续章节会逐步使用它实现线性模型、分类模型、CNN 和 RNN。
- 课程前半部分会先用简单数学模型理解训练过程，再逐渐切换到 PyTorch 的张量、自动求导、数据集封装和神经网络模块。

## 课程主线

本课程可以按下面这条线理解：

1. 先从线性模型开始，理解模型预测值和真实值之间的误差。
2. 再引入梯度下降，理解参数是怎样一步一步被更新的。
3. 接着学习反向传播，理解多层网络中梯度如何传递。
4. 然后使用 PyTorch 重写前面的训练流程，减少手写梯度和参数更新。
5. 最后进入更常见的深度学习结构，包括 Logistic Regression、Softmax、CNN 和 RNN。

## 一个典型训练流程

深度学习训练通常可以拆成下面几步：

1. 准备数据：输入样本 `x` 和目标标签 `y`。
2. 定义模型：给定输入后，模型输出预测值 `y_pred`。
3. 计算损失：用 loss 衡量预测值和真实值的差距。
4. 反向传播：根据 loss 计算参数的梯度。
5. 更新参数：使用梯度下降或优化器调整参数。
6. 重复训练：多轮迭代后，让模型预测结果逐渐接近真实结果。

## 学习建议

- 第一遍学习时，重点理解训练流程，不必急着记住所有 API。
- 后面每章的代码都尽量自己敲一遍，尤其是 `forward`、`loss`、`gradient`、`backward` 这些关键词。
- 遇到 PyTorch 代码时，可以先问三个问题：数据是什么形状，模型输出是什么形状，loss 是怎样算出来的。
- 建议把每章的实验代码、运行结果和自己的问题都放在对应章节文件夹里。
